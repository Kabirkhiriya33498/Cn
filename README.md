This project focuses on early prediction of Type 2 Diabetes using machine learning by comparing four supervised models—Random Forest, Support Vector Machine (SVM), XGBoost, and Logistic Regression—on the Pima Indians Diabetes Dataset. A standardized preprocessing pipeline was implemented, including handling missing values through median imputation, feature scaling, and stratified data splitting to ensure fair evaluation. Each model was optimized using grid search with cross-validation, leading to significant performance improvements, with XGBoost achieving the best results (Accuracy: 88.3%, AUC-ROC: 0.931). The study evaluates models using key metrics such as accuracy, precision, recall, F1-score, and AUC-ROC, highlighting the importance of recall in medical diagnosis to minimize missed cases. Key predictors identified include glucose, BMI, and age, aligning with clinical knowledge. Overall, the project demonstrates that proper hyperparameter tuning and model selection can substantially enhance predictive performance, making machine learning a valuable tool for healthcare decision support.
